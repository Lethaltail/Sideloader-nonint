# Sideloader-nonint

*The missing open-source iOS sideloader fork, with a missing feature.*

Sideloader-nonint is a fork of a project designed to install third-party applications on iOS devices.

This fork has merged code to allow non-interactive logins for use in scripting. 

Apple Account username and password can be set via the environment variables `APPLE_ID_USER` and `APPLE_ID_PWD`.
